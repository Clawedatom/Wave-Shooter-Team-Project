# Unity 2D Alien Survivor - Team Project

Prototype 2D shooter developed by a team of 7 during the first year of university. Players face waves of enemies while navigating procedural maps and aim to survive as long as possible.

My role on this project was **Lead Programmer**.

## Project Overview

This repository contains the complete Unity project for **Alien Survivor**, a 2D shooter game. The game features multiple enemy types with AI, player controls, and procedural maps with obstacles.

## Gameplay

Survive waves of enemies while navigating a map filled with procedural obstacles.

## Key Systems

- Multiple enemy types with AI
- Player character and UI
- Map generation and building system
- Menu and pause system

## Lead Programmer Role

As lead programmer, my responsibilities included:

- Assisting other programmers with complex development tasks
- Offering design ideas and acting as a communication bridge between the design team and programmers
- Integrating systems developed by other programmers
- Fixing issues in areas such as enemy spawn weights, enemy AI, and UI integration
- Contributing to Trello and GitHub dicussions as well as updating them.

## My Contributions

- **Map Generation** - Grid-based tiles with Perlin noise to create rock clusters  
- **Building System** - System for placing turrets  
- **Turrets** - Turrets that target the closest enemy  
- **Systems Development** - Connected various systems throughout the sprint  

## Tools Used
- **Trello** sprint and overall development managed on trello.
- **Discord** logged meeting notes and design ideas.


## How To Play

- **WASD** - Move  
- **Left Click** - Shoot  
- **Escape** - Pause  
- **Tab** - Toggle Building Mode  
- **Right Click** - Place Turret (in Build Mode)

## Reflection
Given more development time, our team planned to enhance the "Alien Survivor" theme with improved assets and animations. During development, we faced an issue with the programmer responsible for the upgrade system, which ultimately led to its removal from the game.

Personally, I would have improved my communication with my teammate and provided more support and oversight when they encountered obstacles they couldn’t overcome.
